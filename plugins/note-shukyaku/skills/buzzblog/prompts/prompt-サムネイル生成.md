# サムネイル生成プロンプト（汎用テンプレート）

> これは骨組みだけの汎用版です。配色やコピーは記事ごとに調整してください。
> 配置先：`<Vault Root>\80_ブログ作成ガイド\プロンプト\サムネイル生成.md`
> モデル：`gpt-image-2`（日本語テキストの描画精度が高い）

## 出力仕様
- 生成サイズ：`1536x1024`
- 後処理：中央の水平帯を 1.91:1 にクロップ → `1280×670`（note推奨）へ縮小
- 形式：PNG

## レイアウトパターン（どれか1つを選ぶ）
- **A：王道スタック** — 中央に大タイトル、上下にサブ要素
- **B：左右分割** — 左にコピー、右にビジュアル
- **C：下段リストパネル** — 上にタイトル、下に要点カード
- **D：ダイナミック** — 斜め構図・大きめの数字で勢いを出す

## プロンプト方針
- 主要なテキスト・ロゴは **画像の中央65%の水平帯に収める**（上下端には重要要素を置かない＝クロップで切れないため）。
- 「薄い色」「パステル」は避け、彩度の高い色（黄／シアン／マゼンタ等）でメリハリを付ける。
- 日本語テキストは正確に、はっきり読めるように描く（"Render Japanese text accurately and clearly readable" を明示）。
- 記事の「サムネ案」セクションがあれば、その配色・コピー指示を踏襲する。

## プロンプトの骨子（例）
```
{レイアウトパターン} の構図。
メインコピー「{日本語タイトル}」を中央に大きく、正確な日本語で描画。
サブ要素「{補足}」。配色は {高彩度カラー}。
主要テキストは中央65%の水平帯に配置。プロ品質のブログサムネイル。
```
