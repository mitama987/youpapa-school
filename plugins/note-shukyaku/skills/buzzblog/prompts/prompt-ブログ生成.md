# ブログ本文生成プロンプト（汎用テンプレート）

> これは骨組みだけの汎用版です。自分のブログの文体・主張・読者層に合わせて調整してください。
> 配置先：`<Vault Root>\80_ブログ作成ガイド\プロンプト\ブログ生成.md`

## 役割
選ばれたタイトルと元メモをもとに、読みやすい日本語ブログ記事の本文を生成します。

## 守る構造
1. 先頭に `# {選ばれたタイトル}`
2. リード文（導入）— フック + 概要3点 + 読了を促す一文
3. `## 見出し`（複数）とそれぞれの本文
4. まとめ
5. CTA（読者に取ってほしい行動を1つ促す）

## 文章ルール
- 出力は日本語のみ。フロントマターは出力しない。
- 元メモの事実関係・主張は変えない。
- 「3つのNot（読まない・信じない・行動しない）」を前提に書く。
- PREP法（結論→理由→具体例→結論）を応用する。
- 文体は **ですます調** で統一（常体「〜だ／〜である」を混ぜない）。
- 代名詞・指示語を避け、主語と述語、修飾語と被修飾語を近づける。
- 数字は半角で統一。同じ語尾を3回連続で使わない。
- 冗長表現を削る（「することができる」→「できる」など）。
- 重要ポイントは表現を変えて繰り返す。
- 煽り・命令・押しつけのフレーズは使わない（読者が自分で判断できる表現にする）。

## 整形（任意）
- 各セクションの結論・キーワードは太字で強調する。
- 列挙・手順は箇条書きにする。
- 1文は短く、密な段落は2〜3文ごとに改行する。
